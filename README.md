Toad Runner is a Unity 6.0 (6000.0.50f1) WebGL project developed for a bachelor‚Äôs dissertation examining whether subliminal visual cues can influence directional choices in a short browser-based game. The project includes both the game itself and the supporting infrastructure required for running the study remotely.

The repository contains:

A lightweight framework for remote data collection using Firebase Realtime Database

Server-side processing and validation through Firebase Cloud Functions

A simple Unity‚ÄìJavaScript WebGL bridge for backend communication

The full game logic used for the experiment, including controlled randomisation and group assignment

The playable study version is available on itch.io:

This setup was designed to support even distribution between experimental groups, minimise opportunities for data manipulation, and synchronise gameplay and survey responses for analysis. The project may serve as a reference for creating similar small-scale behavioural experiments in Unity WebGL environments.
